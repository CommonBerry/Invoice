# Invoice

> Manage your freelance projects elegantly from the terminal

## Warning!
***Invoice is now in Alpha release and may contain bugs, unexpected functioning, missing features or incomplete operating rules***

## About
Invoice is a modern CLI freelance project manager designed to be fast, efficient, and non-distracting.

## Features
- Create and manage projects
- Track budgets and payments
- Don't forget your delivery dates
- [WIP] Beautiful TUI dashboard to manage your projects

## Usage
- `new` - Create a new project
- `list` - List the projects
- `get [options]` - Get a specific project, by id or name
- `start [options]` - Start an existing project, by id (implemented), or name (WIP)
- `done [options]` - Mark a project as finished, by id (implemented), or name (WIP)
- `filter [options]` - Filter started or finished projects
- `delete [options]` - Delete a project, by id (implemented), or by name (WIP)
